## 本地开发

```bash
npm start
```

这个命令会启动本地开发服务器并在浏览器中打开。大多数更改都会实时反映，无需重启服务器。

## 构建

```bash
npm run build
```

此命令将静态内容生成到 `build` 目录中，可以使用任何静态内容托管服务进行部署。

## 部署

```bash
npm run deploy
```

如果你使用 GitHub Pages 进行托管，这个命令是一个便捷的方式来构建网站并推送到 `gh-pages` 分支。
